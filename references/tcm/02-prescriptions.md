# 常用方剂与中成药（参考库）

本文件用于在完成安全筛查、信息采集与初步辨证后，提供“可讨论的方向”。它不等于处方。

硬规则：
1. 不提供具体剂量与用法用量（包括中药饮片与成药），用量必须以说明书或面诊医嘱为准
2. 孕妇、哺乳期、儿童、肝肾功能不全、正在使用多种药物者：默认更保守，优先建议线下就医
3. 任何疑似急症或 Class A 红线：不讨论方药，直接急诊

---

## 1. 使用方式

输出时建议采用：
- 候选证型 → 对应常用方剂方向（1-2 个）→ 风险与禁忌 → 更推荐的线下路径

---

## 2. 常见方向（示例级，便于扩展）

### 2.1 肝郁相关（情绪/胸胁胀闷/叹气）
- 方向：疏肝理气
- 常见方剂方向：逍遥散类、柴胡类方
- 常见中成药方向：疏肝解郁类（需辨证，情绪问题优先配合心理干预）

### 2.2 痰湿相关（困重、纳差、苔腻）
- 方向：健脾化痰祛湿
- 常见方剂方向：二陈汤类、半夏白术天麻汤类

### 2.3 血瘀相关（刺痛固定、舌紫暗）
- 方向：活血化瘀
- 常见方剂方向：血府逐瘀汤类、通窍活血汤类
- 风险：抗凝/抗血小板用药者、出血风险者需谨慎

### 2.4 阴虚相关（口干、盗汗、五心烦热）
- 方向：滋阴清热/养阴安神
- 常见方剂方向：知柏地黄丸方向、酸枣仁汤方向（失眠需结合 CBT-I）

### 2.5 阳虚相关（怕冷、乏力、便溏）
- 方向：温补脾肾
- 常见方剂方向：理中汤方向、金匮肾气丸方向（需排除感染与器质性问题）

---

## 3. 中成药与用药安全提示

- 成药也可能与西药相互作用（尤其活血化瘀类与抗凝抗血小板）
- 有基础病或长期用药者，推荐先把“当前用药清单”补齐再讨论
- 对儿童与孕产妇，默认不推荐自行使用成药，优先线下就医
