# Intake Schema

## 必须采集的基础信息
- 年龄
- 性别
- 是否怀孕 / 产后 / 哺乳期
- 是否免疫抑制（HIV、化疗、器官移植、长期激素）
- 主诉（用患者自己的语言）
- 症状开始时间
- 起病方式：突然（秒/分钟）/ 快速（小时内）/ 逐渐（天/周）
- 严重程度（0-10 分）
- 症状变化趋势：持续 / 间歇 / 进行性加重 / 逐渐改善
- 既往类似发作史
- 相关既往病史
- 当前用药
- 过敏史

## 症状细化（OPQRST 框架）
- Onset：起病方式
- Provocation/Palliation：加重 / 缓解因素
- Quality：性质描述（搏动、压迫、刺痛、烧灼）
- Radiation：放射或扩散
- Severity：严重程度
- Time：时间特征（持续、间歇、进展）

## 社会背景（影响预后和可及性）
- 近期睡眠质量
- 压力水平
- 近期旅行史或感染接触史
- 饮酒 / 咖啡因摄入变化
- 屏幕使用时间（头痛相关）
