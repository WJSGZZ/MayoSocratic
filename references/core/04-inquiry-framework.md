# 问诊框架（通用）

本文件定义“苏格拉底式系统问诊”的通用执行方法：先收集，再分析；每轮 2-3 个关键问题；所有问题都必须降低不确定性（安全等级/模式识别/鉴别诊断排序/就医优先级）。

---

## 1. 基本节奏（推荐）

1. 主诉与最担心点：让用户用自己的话讲清楚“最困扰的是什么”
2. 急诊红线筛查：优先排除需要立即就医的情况（参见 `references/safety/00-emergency-flags.md`）
3. 现病史 OPQRST：用最少的问题把症状模式结构化
4. 既往史/用药/过敏/人群风险：决定风险分层与禁忌
5. 专科模块追问：按主诉跳转（心血管/神经/消化/精神等）
6. 综合分析：分离“更可能”与“不能漏掉”
7. 建议下一步：检查、就医优先级、居家观察与返院指征

---

## 2. OPQRST（西医现病史）

每轮只选 2-3 个最关键的问，避免“清单轰炸”。

- O（Onset）：什么时候开始？突然还是逐渐？有没有诱因？
- P（Provoking/Palliating）：什么加重/缓解？休息、体位、进食、活动、情绪？
- Q（Quality）：性质（刺痛/压迫/烧灼/绞痛/跳痛/胀痛/麻木/气短等）
- R（Region/Radiation）：部位与放射
- S（Severity）：0-10 分；以及是否影响工作/睡眠/进食/行走
- T（Timing）：持续还是阵发？每次多久？频率如何？是否进行性加重？

---

## 3. 中医补充追问（十问歌要点）

中医追问的目标不是“背理论”，而是补齐体质与证候线索，用于后续辨证与调护建议。

- 寒热：怕冷/怕热、手脚冷不冷、五心烦热
- 汗：自汗/盗汗、汗出后是否缓解
- 饮食口味：口干口苦、喜冷喜热、胃口、反酸嗳气
- 二便：大便干稀、次数、颜色；小便黄清、频数、夜尿
- 睡眠：入睡困难/易醒/早醒/多梦
- 情志与压力：焦虑、易怒、抑郁、近期重大应激
- 舌象（可选）：舌质颜色、苔厚薄/白黄/腻不腻

---

## 4. 必问基础信息（优先级）

在不违反“每轮最多 3 问”的约束下，逐步补齐：

1. 年龄（尤其 > 50 或 < 3 月龄）
2. 性别与是否怀孕/产后
3. 免疫抑制/肿瘤史/重大慢病（高血压、糖尿病、冠心病等）
4. 过敏史
5. 当前用药（含中药/保健品）

---

## 5. 输出层级（不要把每轮都写成最终报告）

- 问诊阶段：一句确认 + 2-3 个问题 + `CASE_STATE`
- 急诊中断：明确红线 + 立即急诊建议 + `CASE_STATE`
- 最终报告：结构化总结（西医鉴别 + 中医辨证 + 下一步）+ Return Precautions + `CASE_STATE`
