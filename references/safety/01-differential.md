# Differential Diagnosis Framework

## Core Rule

Never mix probability ranking with emergency-risk ranking.

The final answer must clearly separate:
- What currently fits best.
- What is dangerous to miss.

## 双轴排序原则
鉴别诊断必须按两个维度排序：
- **可能性（Probability）**：基于已收集证据的符合程度
- **危险性（Severity）**：漏诊的后果严重程度

输出分两个区块：

### 区块 A：更符合目前信息
按概率排序，列 1-3 个。

### 区块 B：目前证据不足，但不能漏掉
按危险性排序，列高危诊断。即使概率低，也必须列出并说明触发条件。

对于区块 B 的每一个条目，必须补齐三个句式：
- 触发条件：
- 为什么目前不是最高可能：
- 如果出现该条件怎么办：

## Illness Script 结构（每个疾病的标准条目）

每个候选诊断必须按以下结构进行内部推理（不必全部呈现给用户）：

```
疾病名称
├── 典型人群（年龄、性别、危险因素）
├── 发病机制（一句话）
├── 症状模式（时间线 + 特征组合）
├── 触发因素
├── 支持证据（来自当前对话）
├── 反驳证据（来自当前对话）
├── 尚缺的关键信息
├── 漏诊风险等级（高/中/低）
└── 下一个最有价值的问题
```

## 头痛场景 Illness Scripts

### 紧张型头痛（Tension-type Headache）
- 典型人群：各年龄，压力大、久坐、屏幕使用多
- 机制：头颈部肌肉持续收缩
- 特征：双侧、压迫感/紧箍感、中等强度、不随活动加重、无恶心呕吐
- 触发：睡眠不足、压力、姿势不良、脱水
- 漏诊风险：低

### 偏头痛（Migraine）
- 典型人群：18-44 岁女性，有家族史
- 机制：三叉神经血管系统异常激活
- 特征：单侧（可双侧）、搏动感、中重度、活动加重、伴恶心/畏光/畏声
- 触发：激素变化、睡眠改变、特定食物、压力、咖啡因戒断
- 先兆：视觉先兆（约 30% 患者）
- 漏诊风险：低（本身不危险，但需要与继发性头痛鉴别）

### 蛛网膜下腔出血（SAH）
- 典型人群：40-60 岁，高血压史
- 机制：颅内动脉瘤破裂
- 特征：**雷击样头痛**，数秒内达到极值，「一生最严重」
- 漏诊风险：**极高**（漏诊致死率高）
- 触发条件：一旦突发性+最严重，立即急诊

### 脑膜炎（Meningitis）
- 典型人群：各年龄，近期感染、免疫抑制
- 特征：头痛 + 发热 + 颈部僵硬三联征，可伴皮疹（脑膜炎球菌）
- 漏诊风险：**极高**

### 脑卒中/TIA（Stroke/TIA）
- 典型人群：50 岁以上，高血压、糖尿病、心房颤动
- 特征：突发神经功能缺损（单侧无力、说话困难、视野缺损）伴头痛
- 漏诊风险：**极高**

## 输出格式示例

### 区块 A：更符合目前信息
1. **紧张型头痛** — 双侧压迫感、睡眠不足、无伴随症状，高度符合
2. **偏头痛** — 尚缺少单侧性和恶心证据，可能性中等

### 区块 B：目前证据不足，但不能漏掉
- **蛛网膜下腔出血**：如果出现突然爆发性头痛或「一生最严重」，立即急诊
- **脑膜炎**：如果出现发热+颈僵，立即急诊
- **脑卒中**：如果出现单侧无力/说话困难，立即急诊
